## ES7
- ... 对象合并



## ... 使用  浅拷贝
```js
//原来的对象合并实现 assign()  
let obj = {name:'cometang'};
let ah = {yd:'打游戏',xq:'摄影'};
let obj1 = Object.assign(obj,ah);
console.log(obj1);
// { name: 'cometang', yd: '打游戏', xq: '摄影' }
```
```js
// ... 的方式和assign()效果一样
let obj = {name:'cometang'};
let ah = {yd:'打游戏',xq:'摄影'};
let obj1 ={...obj,...ah};
console.log(obj1);
// { name: 'cometang', yd: '打游戏', xq: '摄影' }
```
