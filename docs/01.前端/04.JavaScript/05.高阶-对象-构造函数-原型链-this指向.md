# 1.构造函数

### 1.1 回顾对象的创建方法

```js
    //通过实例化创建对象
    let user = new Object();
    //对象属性和方法
    user.name = '张麻子'
    user.age = 25
    user.eat = function(){
        console.log('我正在吃饭...')
    }

    //通过字面量创建对象  语法糖
    let user1 = {
        name:'黄四郎',
        age:25,
        eat:function(){
            console.log('我也要吃饭...')
        }
    }

```
