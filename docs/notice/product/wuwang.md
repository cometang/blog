# 勿忘录

 

###### 1. 产品思路

```js
生日勿忘 事务勿忘 特殊日勿忘 
```



```
日期选择：农历+阳历
		
		连续模式：年份+月份+每日 【联系提醒模式为年份的，可设置提前三天 提前一天提醒】
		普通勿忘提醒： 选中具体年月日---选择连续年份/月份
		精确勿忘提醒：	选中具体年月日时间---选择连续年份/月份/每日
```



```js
提醒模式：提前提醒+当日提醒+精确时间提醒 
```



```js
提醒方式：
		邮件（多个邮箱） 
		短信提醒（多个号码/选中号码不能已经被其他账号设置为提醒账号：防止用户恶意批量注册，蹭免费额度）
		可考虑人工智能电话（成本太高）
```



```
变现模式：
	成本计算：邮件免费 短信：1条/0.04元  人工智能电话：未知 （初步估计0.3-0.5元/次）
	
	预计收费：
			邮件提醒免费提供
			短信每年5-10条免费（单个人/家庭够用）
			人工智能电话：预计每天超1元
			
			收费模式：
					普通会员模式：短信提醒免费条数：166条/年  定价：10元/年
					高级会员模式：短信提醒免费条数：1000条/年 定价：50元/年   AI电话： 3次/每年
					
			按量收费：
					短信：10-100条/年     每条 0.1 
						 101-300条/年    每条 0.8
						 300条以上       每条 0.6
					AI电话：1元/条
					
		
        
        后期：广告流量变现
        
        推广：赠送亲人慰问相关产品（化妆品/保健品皆可）
        
```
